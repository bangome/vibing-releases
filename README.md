## Vibing이란?

Vibing은 AI 코딩 도우미(Claude Code)를 누구나 쉽게 사용할 수 있도록 만든 Windows 최적화 데스크톱 앱입니다.

터미널이 낯선 분들도, 코딩이 처음인 분들도 — 아이디어만 있으면 AI와 함께 바로 시작할 수 있습니다.

> Windows 환경에 최적화되어 있습니다. PowerShell 기반 터미널, winget 자동 설치 등 Windows 네이티브 경험을 제공합니다.

## 주요 기능

### 환경 & 프로젝트

- **원클릭 환경 설정** — Node.js, Git, Claude Code를 자동으로 감지하고 설치
- **프로젝트 템플릿** — 웹사이트, React, Next.js, Python 등 다양한 시작점 제공
- **프로젝트 가져오기** — 기존 폴더를 Vibing 프로젝트로 바로 등록
- **프로젝트 설정** — 프로젝트별 환경 변수, 시작 명령어 등 개별 설정 관리
- **인터랙티브 투어** — 처음 사용자를 위한 단계별 기능 안내

### 터미널

- **멀티 터미널** — 여러 터미널을 탭으로 관리, 앱을 재시작해도 세션 유지
- **WebGL 렌더링** — 빠르고 부드러운 터미널 출력
- **하이퍼링크 지원** — 터미널 내 URL 클릭으로 브라우저 열기

### 파일 관리

- **파일 탐색기** — 프로젝트 파일 구조를 사이드바에서 한눈에 확인
- **파일 클립보드** — AI가 작업한 파일을 클립보드에 복사해 다른 곳에 바로 붙여넣기

### 미리보기

- **실시간 미리보기** — 코드 변경사항을 앱 내에서 즉시 확인
- **개발자 도구** — 미리보기 창에서 바로 요소 검사 및 디버깅

### 알림 & 업데이트

- **알림 센터** — 다른 프로젝트의 작업 완료, 오류 등 이벤트를 실시간으로 수신, 해당 프로젝트로 바로 이동
- **자동 업데이트** — 새 버전이 출시되면 앱 내에서 바로 업데이트

### UI / 테마

- **다크/라이트 테마** — 눈이 편한 환경에서 작업

## 다운로드

[Releases](https://github.com/bangome/vibing-releases/releases) 페이지에서 최신 버전을 다운로드하세요.

| 플랫폼 | 파일 |
|--------|------|
| Windows | `Vibing-Setup-x.x.x.exe` |

> macOS, Linux 지원은 추후 예정입니다.

## 시작하기

1. 설치 파일을 다운로드하고 실행합니다
2. 온보딩 가이드를 따라 개발 환경을 설정합니다 (Node.js, Git, Claude Code 자동 설치)
3. Anthropic API 키를 입력합니다
4. 새 프로젝트를 만들고 AI와 대화를 시작합니다
